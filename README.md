# Zest 🏃

> *Run it up.*

Zest is an iOS running app that makes your music react to how you move. Speed opens up the audio filter in real time. Break your stride, stumble, or stop suddenly — and the track distorts until your rhythm comes back.

---

## Features

- Music filter driven by GPS speed
- Gait anomaly detection via accelerometer and gyroscope
- Live speedometer with distance and average speed
- Genre browser and custom track import
- Session history
- Lock screen notification while running

---

## Stack

SwiftUI · AVAudioEngine · CoreMotion · CoreLocation · CoreData · Combine

---

## Requirements

- Xcode 15+
- iOS 17+
- Physical device
